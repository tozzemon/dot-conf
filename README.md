# Wayland setup

Hyprland, Foot, Rofi, Waybar, Qutebrowser, Neovim, Vifm, Mako.

```bash
git clone https://github.com/tozzemon/dot-conf.git && cd dot-conf && mkdir -p ~/.config && cp -r scripts hypr foot rofi waybar qutebrowser nvim vifm mako ~/.config
```
